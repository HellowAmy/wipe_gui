(1)简介：
	linux下的wipe工具是用于粉碎文件的工具，大致原理为反复用随机数反复覆盖原文件磁盘段35次。wipe工具用于消除数据并保护数据安全有重大意义，因为普通的右键删除并不是真正意义上的删除，只是将文件标记取消，实际内容还在，可以被轻易恢复。wipe工具只提供命令行操作，如果需要频繁使用有诸多不便，本程序将提供一个功能基于wipe的图形界面。

(2)wipe源码地址：https://github.com/berke/wipe
